# MandiSense UI Wireframes: Mobile-First Design for Collective Intelligence

## Design Philosophy

MandiSense interface design follows a fundamental principle: **the technology should disappear into the negotiation**. Every visual element serves comprehension, not persuasion. The interface becomes invisible infrastructure that enhances human interaction without disrupting natural market behavior.

The design prioritizes:
- **Cognitive Minimalism**: Essential information only
- **Cultural Sensitivity**: Appropriate for Indian market contexts
- **Accessibility**: Functional across literacy levels
- **Environmental Adaptation**: Optimized for outdoor market conditions

### Design Elements

**Header Section**
- **MandiSense branding**: Minimal, confidence-inspiring
- **Subtitle**: "Collective Market Memory" establishes purpose
- **Status indicators**: Battery, network for outdoor reliability

**Commodity Selection**
- **Visual commodity icons**: Immediate recognition
- **Quantity specification**: Clear transaction scope
- **Dropdown simplicity**: Minimal cognitive load

**Translation Interface**
- **Speaker identification**: Role-based, not personal
- **Language indicators**: Clear linguistic context
- **Original text display**: Preserves authenticity
- **Translation confidence**: Visual quality indicators
- **Real-time feedback**: Immediate linguistic bridge

**Interaction Controls**
- **Primary action**: Voice input (natural for oral markets)
- **Secondary option**: Text input (accessibility)
- **Large touch targets**: Outdoor usability

**Privacy Assurance**
- **Explicit privacy statement**: Builds trust
- **No recording emphasis**: Addresses surveillance concerns
- **No ID tracking**: Reinforces anonymity

### Accessibility Features
- **High contrast ratios**: Outdoor visibility
- **Large text sizes**: Multi-generational usability
- **Icon-text combinations**: Cross-literacy support
- **Voice-first interaction**: Accommodates limited literacy

### Design Elements

**Header Context**
- **Purpose clarification**: "Live patterns, not recommendations"
- **Information framing**: Context, not prescription

**Price Visualization**
- **Range display**: Visual spectrum, not point values
- **Current position**: Contextual placement
- **No optimization**: Avoids "correct price" implications

**Market Intelligence Metrics**
- **Market Coherence**: Collective intelligence level
- **Trust Density**: Aggregate trust patterns
- **Activity Level**: Market participation context
- **Pattern Stability**: Behavioral consistency measure

**Metric Presentation**
- **Numerical precision**: Three decimal places for sophistication
- **Qualitative labels**: "Developing," "Moderate" for accessibility
- **Dual information**: Numbers + human-readable descriptions

**Ethical Disclaimer**
- **Explicit limitation**: No prescriptive guidance
- **Pattern emphasis**: Collective behavior focus
- **Anonymity reminder**: Reinforces privacy protection

### Information Architecture Principles
- **Context over recommendation**: Provides awareness, not direction
- **Collective over individual**: Aggregate patterns, not personal data
- **Descriptive over prescriptive**: Observational, not instructional

### Design Elements

**Completion Confirmation**
- **Success indicator**: Clear transaction closure
- **Deal summary**: Price, quantity, total for verification
- **Achievement framing**: Positive completion experience

**Market Alignment Signal**
- **Central prominence**: Primary interface focus
- **Contextual language**: Reflective, not prescriptive
- **Collective framing**: Market patterns, not individual judgment
- **Empowering tone**: Validates positive behavior

**Signal Metrics**
- **Pattern Familiarity**: How well-recognized this behavior is
- **Signal Confidence**: System certainty in guidance
- **Transparency**: Shows algorithmic reasoning
- **Dual presentation**: Numerical + qualitative

**Market Context Data**
- **Similar deals**: Provides comparative context
- **Market consistency**: Indicates pattern stability
- **Range information**: Contextualizes final price
- **Aggregate focus**: No individual comparisons

**Ethical Explanation**
- **Purpose clarification**: Collective behavior reflection
- **Privacy assurance**: No permanent records
- **Anonymity reminder**: No identity tracking
- **Temporal limitation**: Transaction-specific guidance

### Signal Language Principles
- **Observational**: "This approach aligns with..." not "You should..."
- **Collective**: "Market patterns" not "Other users"
- **Empowering**: Validates positive behavior
- **Non-punitive**: Even deviations are contextual, not critical
