# MandiSense 2.0 - Requirements File

## System Requirements

### Minimum System Requirements
- Operating System: Windows 10, macOS 10.15, Ubuntu 18.04 or later
- RAM: 4GB minimum, 8GB recommended
- Storage: 2GB free space
- Internet Connection: Broadband connection for real-time features

### Development Environment
- Node.js: 16.0.0 or higher
- npm: 8.0.0 or higher
- Git: 2.30.0 or higher
- Code Editor: VS Code, WebStorm, or similar

## Browser Requirements

### Supported Browsers
- Google Chrome: 90.0 or higher ✅
- Mozilla Firefox: 88.0 or higher ✅
- Safari: 14.0 or higher ✅
- Microsoft Edge: 90.0 or higher ✅

### Required Browser Features
- JavaScript ES2020 support
- Web Speech API (for voice features)
- Local Storage API
- File API (for image uploads)
- Geolocation API (for location features)
- WebRTC (for real-time features)

## Hardware Requirements

### For Voice Features
- Microphone: Built-in or external microphone
- Speakers/Headphones: For voice output
- Camera: For image capture (disease detection)

### For IoT Features (Optional)
- IoT Sensors: Soil moisture, temperature, humidity
- Network Connection: WiFi or cellular for sensor data
- Gateway Device: For sensor data aggregation

## Network Requirements

### Bandwidth
- Minimum: 1 Mbps for basic features
- Recommended: 5 Mbps for optimal experience
- Voice Features: Additional 0.5 Mbps
- Video/Image: Additional 2 Mbps

### Ports
- HTTP: 80 (for API calls)
- HTTPS: 443 (for secure connections)
- WebSocket: 8080 (for real-time updates)

## Security Requirements

### HTTPS
- SSL/TLS certificate required for production
- Secure API endpoints
- Encrypted data transmission

### Data Privacy
- Local data storage (no server transmission)
- User consent for data collection
- GDPR compliance for international users

## Installation Requirements

### Prerequisites Check
```bash
# Check Node.js version
node --version  # Should be 16.0.0+

# Check npm version
npm --version   # Should be 8.0.0+

# Check Git version
git --version   # Should be 2.30.0+
```

### Installation Steps
1. Clone repository
2. Install dependencies: `npm install`
3. Configure environment variables
4. Start development server: `npm start`
5. Build for production: `npm run build`

## Troubleshooting

### Common Issues
1. **Node version mismatch**: Use nvm to manage Node versions
2. **npm install fails**: Clear npm cache and retry
3. **Voice features not working**: Check browser permissions
4. **Build fails**: Check TypeScript errors and fix
5. **Performance issues**: Check browser developer tools
